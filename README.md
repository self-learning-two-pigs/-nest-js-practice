## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Start database
```bash
# start docker
$ colima start

# start database services
$ docker-compose up -d

# check database services
$ docker-compose ps
$ docker ps

# mysql management page
http://localhost:8080/
```
